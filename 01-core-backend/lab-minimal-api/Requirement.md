# Lab: Minimal API with EF Core

## Objectives

- Build a CRUD REST API using ASP.NET Core Minimal API syntax
- Connect to PostgreSQL using EF Core with proper migrations
- Implement input validation, error handling, and OpenAPI documentation
- Apply best practices: no-tracking queries for reads, async throughout

## Tasks

- [ ] Create .NET 9 project with Minimal API

- [ ] Define entity and DbContext
- [ ] Implement GET / POST / PUT / DELETE endpoints
- [ ] Add FluentValidation for request validation
- [ ] Generate OpenAPI spec with Swagger UI
- [ ] Write integration tests with WebApplicationFactory

## Expected Output

Running API with Swagger UI at `/swagger`, all endpoints tested, migration script committed.
