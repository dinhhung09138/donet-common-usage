# Lab: Calendar & Scheduler Component (Vue)

## Objectives

- Build or integrate a month/week/day calendar view showing events from an API
- Implement create/edit/delete of events via click-and-drag time-slot selection
- Handle recurring events and timezone-correct rendering
- Handle overlapping events with a readable stacked layout

## Key Concepts

`FullCalendar (vue3) / vue-cal` · `date-fns/Luxon timezone handling` · `drag-to-create event` · `recurrence rules (RRULE)` · `overlap layout`

## Tasks

- [ ] Integrate a calendar library configured with month/week/day views
- [ ] Load events from a mock API and render them with correct local-timezone conversion
- [ ] Implement drag-to-select a time slot to open a 'create event' form pre-filled with that range
- [ ] Support editing an event's time by dragging/resizing it on the calendar
- [ ] Add a simple recurrence option (e.g. 'repeats weekly') and expand it into individual occurrences for display
- [ ] Verify overlapping events render side-by-side without visually colliding

## Expected Output

A working month/week/day calendar with drag-to-create/resize events, correct timezone handling, and readable overlap layout for double-booked slots.
