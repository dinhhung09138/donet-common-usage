# Module 6 — AI Tooling & Agentic

**Timeline:** Month 5–8 (parallel with Module 3/4)

Labs covering Azure OpenAI, Semantic Kernel, Anthropic Claude API, RAG pipelines, agentic patterns, MCP servers, and GitHub Copilot workflow.

## Labs

| Lab | Description |
|-----|-------------|
| `lab-azure-openai-dotnet` | Azure OpenAI SDK: chat completions, streaming, function calling, embeddings |
| `lab-semantic-kernel-basics` | Semantic Kernel: kernel setup, native plugins, prompt functions, chat history |
| `lab-semantic-kernel-rag` | Semantic Kernel with memory and Azure AI Search vector store |
| `lab-claude-api-tooluse` | Anthropic Claude Messages API with tool use and streaming in .NET |
| `lab-rag-pipeline` | End-to-end RAG: document chunking, embedding, vector search, grounded LLM answer |
| `lab-ai-agent-agentic-loop` | Build agentic loop: multi-step tool calling with stopping conditions |
| `lab-mcp-server-dotnet` | Build MCP server in .NET exposing custom tools to Claude and Cursor |
| `lab-copilot-workflow` | GitHub Copilot prompting strategies and AI-assisted development workflow |

Each lab is a standalone runnable project. See `PREPARATION_PLAN.md` for full knowledge areas and interview questions.
